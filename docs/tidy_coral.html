Tidy Coral

Introduction

We are going to spend the morning beginning a tidy coral analysis that you will continue on your own in the afternoon. We’ll be using what we learned in the data-science-training but with real coral data from Oahu.

Our plan is to combine benthic observation data with buoy oceanographic data and explore patterns.

The benthic data comes from digital still images through the Hawaii Coral Reef Assessment and Monitoring Program (CRAMP), and the buoy data is from the Kaneohe Bay NOAA buoy at Coconut Island (Station MOKH1).

We’ll also be using a few new R packages that are super helpful: stringr, janitor, and skimr.

Wrangle

We’ve got buoy_raw as the raw data we read from online. Let’s create a new variable called buoy that we’ll wrangle instead of that raw data (especially nice if you’ve got poor internet and don’t want to read it in each time!)

buoy <- buoy_raw

Let’s look at the column headers.

##  [1] "#YY"  "MM"   "DD"   "hh"   "mm"   "WDIR" "WSPD" "GST"  "WVHT" "DPD" 
## [11] "APD"  "MWD"  "PRES" "ATMP" "WTMP" "DEWP" "VIS"  "TIDE"

We’ve got two rows of information about what the data represent. R thinks that the first row are the column headers, and it considers the second row data. Let’s clean up those names. But we don’t want to lose either of those rows, because they both have important and unique information (measurement and units).

So, let’s see if we can take that the first row of data (the units) and stick it on the with the column names (measurement). Then, we can get rid of that units row.

In the stringr package, there is a way to combine strings using the str_c function.

There’s 3 things we want to do to these column names:

  1. make the column header a combo of rows 1 and 2
  2. clean up the header; get rid of # and /
  3. delete the now-redundant row 1
## 1. overwrite column names
names(buoy) <- str_c(names(buoy),                  ## first thing to combine
                  buoy %>% filter(`#YY` == "#yr"), ## second thing
                  sep = "_")                       ## separate by this

## inspect
names(buoy) ## Looks a lot better

## [16] "DEWP_degC" "VIS_nmi"   "TIDE_ft"
## 2. clean up a bit more to get rid of the `#`s and the `/`s. 
names(buoy) <- str_replace_all(names(buoy), "#", "")  # replace `#` with nothing (`""`)
names(buoy) <- str_replace_all(names(buoy), "/", "_") # replace `/` with `_`

Date.

OK let’s look at these dates — looks like we’ll have to clean them up. Right now the dates are across several columns. Let’s combine them into dates.

## first change to numeric
buoy <- buoy %>%
  mutate(YY_yr = as.numeric(YY_yr), 
         MM_mo = as.numeric(MM_mo), 
         DD_dy = as.numeric(DD_dy))

## pad zeros (add this after join attempt below)
buoy <- buoy %>%
  mutate(MM_mo = str_pad(MM_mo, 2, "left", pad = 0),
         DD_dy = str_pad(DD_dy, 2, "left", pad = 0))

Now we unite the columns:
